The folder includes all alternative header (horizontal) navigations for the application "texts".

1. Files that include _Wide are wide menus including all categories and subcategories, if any

2. Files that include _By... are drop-down menus including all categories and subcategories, if any.


3. The file "functions_Nav_Queries.php" includes all possible queries 
and is included in all menu alternatives.

4. The file "functions_Nav_Menu_List.php" includes all possible functions 
that create the alternative manu links and is included in all menu alternatives.

The Applications Menu with all additional applications that can be included in any site design.

It is a separate, independed menu, the can be placed: 
-   on the top of the page, above Header and the main navigation menu (sxNav_Apps_on_Top.php).
-   in the same container as the logotype and the main navigation menu,
    placed above the main navigation menu (sxNav_Apps_in_Nav.php).

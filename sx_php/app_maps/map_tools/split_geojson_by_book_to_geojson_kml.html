Split a GeoJson file with Places in Pausanias' 10 Books to 10 GeoJson and KML Files

The GeoJson file must contains the Property Name passages or Passages, with pre-structured values for Books, Chapters and Sections in JSON format.

The GeoJson file is filtered by book and split to:

  • 10 GeoJson files, one for each book.
  • 10 KML files with the element ExtendedData that includes property names and values within Data and Value elements.
  • 10 KML files with the element Description that includes property names and values in HTML format. The name of these files has the suffix _HTML.

The folder includes all alternative main (vertical) navigations for the application "texts".

1. Files that include _Acc_ are accordion menus including pubished texts as final links
for groups, categories or subcategories

2. Files that include _By... are main (vertical) menu alternatives to header menus, 
with the group, category or subcategory as the final link.

s
1. The file "functions_PublishedTexts_Queries.php" is included in 
the default header (defaultHeader.php) and includes all qeries for all
alternative navigations for published texts (_Acc_ files).

2. The _By... files get their values from function in the file sxNav_Header/functions_Nav_Queries.php,
which is also included (once) in the header (horizontally) navigation files (in sxNav_Header).

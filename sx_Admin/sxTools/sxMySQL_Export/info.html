How to use this Application

  • You use this application to save Database Tables as Files in the Default Private Folder of the Remote Server - or to download them in your Local System.
  • You can then use these files with the application Import to MySQL in two different ways:
    1. As Backups to restore the corresponding Database Tables to there previous state. In that case, use the Import Type Truncate and Insert. Remember to first set you website in Update Mode, to avoid errors in the site when a table is trancated, and then change the website back to Running Mode.
    2. You can change the content of a file (update existing records and add new ones) and then update the existing Records of the corresponding Database Table and insert new ones.
  • If you only want to backup your database tables, consider using the alternative applications Backup Small Database or Backup Big Database.

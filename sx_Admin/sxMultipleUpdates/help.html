1. In the form Select Table and Fields,
    • Select the table you want to update
      • The Primary Key of the Table and all its fields will appear automatically
    • Copy and Pastate the Primary Key Name, the Field Name of the field that you want to Update, and the Field Names of any other fields that you like to view before update. Separate field names by a comma (,).
    • Click on Get Records to get the current values of the selected fields
  2. In the form Update a Table Field, write the Field Name of the field that is to be Updated (the Table Name and the Promary Key are entered automatically).
    • Define the Old Value of the field that will be updated and the New Value that will replace it.
    • Optionally, you can also define the Name and Value of a Control Field, usually a Group ID, Category ID or Subcategory ID.
      • The Control Field is added as a WHERE clause in the UPDATE statemen: only records where the Control Field is equal to the defined value will be checked for update.
      • If you do not define the Name and Value for a Control Field, all Current Values of the selected Field Name to be updated will be replaced with the New Value.
    • Click on Check Affected Fields Before Update to view a list with all records that will be affected by uppdating.
    • Click on Update Fields for final update.
  3. You can always reverse the update process - with the risk, however, that old values that already were equal to the new one will also be reversed.
